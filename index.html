Berkeley Crime Data¶

Let's take a look at some crime data from Berkeley, Ca! I wanted to create a simple notebook for basic data exploration and to get some play with GeoPandas. This notebook will explore crime data and try to derive some basic conclusions from the analysis.

The data comes from https://data.cityofberkeley.info/browse, which has a variety public data for Berkeley. It include maps and public information such as police data, energy usage etc...

To run this notebook you'll need berkeley block group maps as shapefiles and the most recent Berkeley-PD calls for services as a csv.

crime_data = pd.read_csv('Berkeley_PD_-_Calls_for_Service.csv')

def get_point(loc_str):
    try:
        lat, lng = map(float, loc_str.split('\n')[-1][1:-2].split(','))
        return Point(lng, lat)
    except ValueError:
        return Point([-1,-1])

crime_data.rename(columns={'CASENO':'Case Number', 
                           'OFFENSE': 'Offense', 
                           'EVENTDT': 'Event Date',
                           'EVENTTM': 'Event Time',
                           'CVLEGEND': 'Crime Legend',
                           'CVDOW':'Day of Week',
                           'InDbDate':'Database Date',
                           'Block_Location': 'Location',
                           'BLKADDR': 'Block Address'}, inplace=True)

crime_data['Coord'] = crime_data['Location'].map(get_point)
crime_data = crime_data.drop('Location', axis=1)

Lets plot the crimes on a map¶

Lets now plot these car burglaries on the map of Berkeley. To do this we use Geopandas. First we've instantiated a GeoDataFrame with the crime_data panadas DataFram from above. We need to tell the GeoDataFrame which column contains the Geo data (geometry). This is why we created the Point geometries for the crimes using their lat and long.

We can also read in the shapefiles that have the polygons describing the boundaries of block groups.

One last note: Every GeoDataFrame needs to have a Coordinate Reference System (crs) set. There are thousands of these but the moset comonly used one is epsg:4326.

In [8]:
geo_crime_data = geopandas.GeoDataFrame(crime_data, geometry='Coord')
geo_crime_data.crs = {'init': 'epsg:4326'}
geo_crime_data = geo_crime_data[geo_crime_data.Coord != Point(-1,-1)]

berkeley = geopandas.read_file('Census Block Group Polygons 2010/geo_export_c2d6150e-03a5-4b10-a7e1-72d684674105.shp')
berkeley.crs = {'init': 'epsg:4326'}

fig, ax = p.subplots(1, figsize=(10,10))
berkeley_map = berkeley.plot(ax=ax, color='none', lw=1)
geo_crime_data.plot(ax=berkeley_map, marker="o", markersize=10, color='red', alpha=0.3, edgecolor='black')
